# ISTQB CT-AI v2.0 Interactive Study Guide

Practice-first, fully static learning guide for the **ISTQB Certified Tester AI Testing (CT-AI) v2.0** certification. No login, no backend — progress saved in your browser.

## Run locally

```bash
npm install
npm run dev   # http://localhost:3000
```

## Deploy (Vercel)

```bash
npx vercel    # no env vars needed
```

## What's inside

- 7 chapters · 50 lessons · all 43 LOs + 7 hands-on exercises
- Concept → Quiz → Interactive Exercise → Complete (practice-first)
- 46-question mock exam (official sample v2.1) with timer + results
- 168-term glossary, searchable by chapter
- 100% static — zero backend, zero auth
